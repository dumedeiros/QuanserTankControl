FusionWidgets XT

How to Install

Installing FusionWidgets XT is a simple copy-n-paste procedure for web applications.

  1. Open Charts folder where all the charts (SWF and JavaScript files) are located
  2. Copy the entire Charts folder to the root of your web application

That's it! Now you are only 15 minutes away from your wow moment.
