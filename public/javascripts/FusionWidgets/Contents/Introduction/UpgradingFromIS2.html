Upgrading from Instrumentation v2 to FusionWidgets XT

This section is meant for users who have implemented Instrumentation v2 in their applications and are now upgrading to FusionWidgets XT.

Due to the extensive set of new features introduced in FusionWidgets XT, it's not 100% backward compatible with Instrumentation v2 on the cosmetics side. On the functional side, FusionWidgets XT is mostly backward compatible. That means, when you replace your Instrumentation v2 SWFs with FusionWidgets XT SWF files, you'll find that the chart functions correctly, but there will be visual changes between your original and new chart.

To upgrade from Instrumentation v2 to FusionWidgets XT charts, you'll need to undertake the following:

  1. Replace v2 SWF with the new XT SWF Files (present in Download Package > Charts folder).
  2. Update your code to use new SWF names.
  3. Update your code to use the FusionCharts JavaScript Class (copy and use all JavaScript files form Download Package > Charts folder).
  4. Clear your browser cache and view the chart.
  5. You might now find that your charts aren't looking the way they used to. You'll have to tweak some attributes to get back to v2 look.
  6. To use the new features of XT charts, you'll have to add the required attributes/elements to the XML document.
