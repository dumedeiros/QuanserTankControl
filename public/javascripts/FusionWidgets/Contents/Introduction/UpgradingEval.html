Upgrading from Trial

This section is meant for users who have implemented the Trial version of FusionWidgets XT in their application and are now upgrading to the commercial license.

There is no feature limitation/difference between the trial version and the commercial version of FusionWidgets XT, except for the fact that the trial version charts are watermarked. If you have been using the trial of FusionWidgets XT in your application and are now upgrading to a commercial license of FusionWidgets XT, you'll need to do the following:

  1. Replace the Trial version Flash (SWF) files with the newly purchased Flash (SWF) files (present in Download Package > Charts folder)
  2. Replace the Trial version JavaScript files with the newly purchased JavaScript files (present in Download Package > Charts folder)
  3. If you were using a version earlier to FusionWidgets XT, make sure you have copied FusionCharts.HC.js and FusionCharts.HC.Widgets.js and jquery.min.js (along with FusionCharts.js) from the Licensed Download Package > Charts folder
  4. Clear your browser cache - very important step

In case, you are also using charts from FusionCharts XT or earlier version, you need to download or update to the latest FusionCharts pack and copy FusionCharts.HC.Charts.js from the Charts folder of the downloaded FusionCharts pack.

Moreover, if you are using charts from PowerCharts v3.2 or earlier version, you need to download or update to the latest PowerCharts pack and copy FusionCharts.HC.PowerCharts.js from the Charts folder of the downloaded PowerCharts pack.

When you now view your application, you won't see any watermarks on the charts.
