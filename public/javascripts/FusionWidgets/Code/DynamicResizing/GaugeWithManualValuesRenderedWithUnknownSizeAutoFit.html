Dynamic Resizing

The sample shows a gauge which was configured using the original width as 260 px and original height as 200 px. The gauge is placed in a container whose dimensions are generated randomly on each page refresh. Moreover, the original width and height are passed to the gauge using <chart origW="260" origH="200" ...>
