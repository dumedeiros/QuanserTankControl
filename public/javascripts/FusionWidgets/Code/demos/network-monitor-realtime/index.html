Real-time Network monitoring demo

 

The network monitoring dashboard below shows the real-time status of a network router. The real-time load of each line of the router is shown by the bulb gauges above each line. The operation capacity is shown by a cylinder on the right. The cylinder and all the bulbs with connection update every second with real-time values.

NOTE: Random data is used to update the gauges in this demo.
